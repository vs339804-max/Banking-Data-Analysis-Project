# Banking Financial Analysis & Interactive Dashboard

This project provides a 360-degree view of banking data using Python for analysis and Power BI for visualization.

## ✨ Project Highlights
- **Data Source:** Automated data fetching from a MySQL database.
- **Python Analysis:** Conducted EDA, identified correlations between loans and deposits, and performed customer segmentation.
- **Interactive Dashboard:** Built a Power BI report with KPIs for total balances, risk assessment, and demographic distribution.

## 🛠️ Tools Used
- **Languages:** Python (Pandas, Seaborn, Matplotlib), SQL
- **BI Tool:** Power BI (DAX, Interactive Slicers)
- **Database:** MySQL

## 📈 Insights Identified
- Identified "Credit-Dependent" vs "Stable" customer segments.
- Found that income levels do not strictly dictate saving habits, highlighting the need for behavioral targeting.
